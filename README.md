# batch-csv-files-in-nodejs

Process csv files with nodejs and PLS

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Running

```sh
node dist/index.js
```

## Testing

To run unit tests:

```sh
npm test
```
